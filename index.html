From Gaussian Noise to Molecules — Atom Types and Coordinates in One Shot

One ODE to Rule Them All: Joint Discrete-Continuous Molecular Generation

Current molecular generative models treat discrete atom types and continuous 3D coordinates as separate problems — coupling a CTMC-based discrete flow with a continuous flow matcher, each with its own velocity field, its own integration schedule, and compounding factorisation errors at low step counts. We take a different path. By embedding atom types as one-hot vectors in \(\mathbb{R}^K\) and concatenating with 3D coordinates, we construct a single joint state \(\mathbf{y} \in \mathbb{R}^{n \times (K+3)}\) and learn one unified velocity field that transports isotropic Gaussian noise to molecular data. One ODE. One denoiser. No CTMC machinery. Discrete atom types emerge naturally via argmax at \(t = 1\), while coordinates are read out directly. Algorithm 3 Full Theory

Joint State Space. Consider a molecule of n atoms. Each atom i has a discrete type \(c^i \in \{1, \dots, K\}\) (here \(K = 3\): C, N, O) and a 3D coordinate \(\mathbf{x}^i \in \mathbb{R}^3\). We embed both modalities into a single Euclidean space by concatenating one-hot atom types with coordinates:

\[\mathbf{y} = [\,\mathbf{e}\;;\;\mathbf{x}\,] \in \mathbb{R}^{n \times (K+3)}\]

The Generative ODE. Starting from pure Gaussian noise \(\mathbf{y}_0 \sim \mathcal{N}(\mathbf{0}, \mathbf{I})\), we integrate a learned velocity field that pushes the noisy joint state toward clean molecular data. At each time \(t\), a denoiser \(D_\theta(\mathbf{y}_t, t)\) predicts the clean target, and the velocity is:

\[\dot{\mathbf{y}}_t = b_t(\mathbf{y}_t) = \frac{D_\theta(\mathbf{y}_t,\, t) - \mathbf{y}_t}{1 - t}\]

Both the discrete embedding (atom type probabilities) and the continuous coordinates evolve under this same field simultaneously. The cross-modal coupling is the key advantage: atom type identity influences where coordinates flow, and spatial context informs which atom types emerge.

Decoding. At \(t = 1\), we split the terminal state \(\mathbf{y}_1 = [\,\hat{\mathbf{y}}^{\text{disc}}\;;\;\hat{\mathbf{y}}^{\text{cont}}\,]\). Discrete atom types are decoded via argmax: \(\hat{c}^i = \mathrm{argmax}_k\;\hat{\mathbf{y}}^{\text{disc}}_{i,k}\). Coordinates are read directly: \(\hat{\mathbf{x}}^i = \hat{\mathbf{y}}^{\text{cont}}_i\). The result is a complete molecule — atom types and 3D geometry — generated by a single ODE in one unified space.
